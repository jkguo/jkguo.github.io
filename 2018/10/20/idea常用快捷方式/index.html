idea常用快捷方式

Oct 20, 2018

最后更新日期：2018/10/20

[toc]

帮助

快捷键 描述 备注
ctrl + shift + A 查找动作 idea的帮助

跳转

快捷键 描述 备注
ctrl + alt +] 下一个窗口 同时打开多个项目时使用
ctrl + alt +[ 上一个窗口 同时打开多个项目时使用
ctrl + E 显示最近打开的文件 -
ctrl + shift + E 显示最近修改的文件 -
ctrl + alt + 向左箭头 跳到上一次光标停留的地方 -
ctrl + alt + 向右箭头 跳到下一次光标停留的地方 -
ctrl + shift + Backspace 跳到上一次修改的地方 -
F11 添加标签 打开收藏夹（alt + 2），可查阅标记了标签的代码行
shift + F11 bookmarks窗口查看标记了标签的代码行 -
alt + F11 添加标签码 弹出小窗口，选择0-9,A-Z作标记码
ctrl + shift + 0~9 添加标签号 -
ctrl + 0~9 根据对应的标签号跳到对应的代码行 -

代码搜索

快捷键 描述 备注
ctrl + N 类搜索 Navigate -> Class
ctrl + shift + N 文件搜索 include non-project file选项，是否搜索非本项目的文件
ctrl + alt + shift + N 符号搜索 全局方法名和变量名搜索
ctrl + shift + F 字符串搜索 全局搜索

代码编辑

快捷键 描述 备注
ctrl + shift + U 字符串大小写切换 -
ctrl + 向右箭头 跳到当前单词的结尾 -
Home 跳到当前行的头 -
End 跳到当前行的尾 -
alt + Insert 代码生成 如：get、set 等
ctrl + J 插入live template -
ctrl + alt + T 围绕代码编码 如：if、try/catch、while 等
alt + enter 智能修复 -
F2 跳转到代码错误的位置 -

批量操作

快捷键 描述 备注
alt+ J 选中下一个与当前字符串相同的 批量操作
alt+ shift + J 撤销最后选中 与上一个命令配合使用
ctrl + alt + shift + J 选中所有与当前字符串相同的 -
ctrl + alt + shift + 鼠标左箭 选择批量操作的位置 可任意代码行

代码重构

快捷键 描述 备注
shift + F6 重命名 -
alt + F6 当前选中的方法重构 -
ctrl + alt + V 抽取数据改为变量 -
ctrl + alt + C 抽取数据改为静态常量 -
ctrl + alt + F 抽取数据改为成员变量 -
ctrl + alt + P 抽取当前的变量改为方法变量 -
ctrl + alt + M 抽取当前代码改为独立的方法 -

文件操作

快捷键 描述 备注
ctrl + alt + N 新建文件 -
F5 复制文件 -
F6 移动文件 -
ctrl + C 点中文件，复制文件名在项目中目录中，选中文件 -
ctrl + shift + C 点中文件，复制文件的绝对路径 在项目中目录中，选中文件
ctrl + alt + shift + C 点中文件，复制文件的相对路径 在项目中目录中，选中文件
ctrl + alt + V 打开粘贴板 -
